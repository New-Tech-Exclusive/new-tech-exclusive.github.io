Pyrix Editor

A text editor built with, and optimized for Python. Featuring native python support, LazyVim like UI and Neovim-style keybindings, and a hyper-minimalist aesthetic inspired by modern TUI workflows.

VYL-lang Language

A compiled, Python-like language with C-level control. Designed to replace C/C++ with a cleaner syntax while maintaining zero-cost abstractions and native x86_64 efficiency.

QIEL AI Arch

Quantum-Inspired Entanglement Lattice. A non-traditional AI architecture exploring deep learning patterns modeled after quantum entanglement. This isn't another Transformer. (Status: Experimental / WIP, working on a training script)
